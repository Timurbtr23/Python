Инструкция по Установке и Использованию голосового ассистента Эмили.

1. Чтобы запустить программу - запустите файл assistant.pyw
2. Чтобы закрыть программу - нажмите на крестик.
3. Чтобы свернуть программу - нажмите на кнопку свернуть.
4. Чтобы программа начала вас слушать - нажмите на кнопку Слушать Команду. Ждите пару секунд.

Важно! Для работы программы компьютер должен иметь микрофон.
Важно! Для работы программы пользователь должен установить на свой компьютер Python .
	Ссылка на скачивание - https://www.python.org/ftp/python/3.7.1/python-3.7.1.exe
	Инструкция по установке - https://youtu.be/IU4-19ofajg?t=152

Примечание! Для корректной работы программы, нужно установить следующие библиотеки через pip install
Открываете Командную строку и пишете следующие команды:
pip install urllib
pip install PyQt5
pip install SpeechRecognition
pip install gtts
pip install pygame 
pip install html2text
pip install webbrowser 

Приносим извинения за неудобства в установке программы. В будущих релизах постараемся упростить процесс установки.
